### Advertencia para el punto 6
- Si no es posible visibilizar las imagenes de las banderas, es probable que el problema se deba a la carga de origen cruzado (CORS). De ser este el caso, deberás:
    - Opcion 1:
        1. Instalar algún software que permita crear un servidor web como XAMPP, que trae configurado Apache.
        2. Clonar el proyecto dentro de la carpeta "htdocs" de XAMPP. Si la instalación previa fue correcta seguir los siguientes pasos:
            - En Windows dirigirse a:
                ```bash
                C:\xampp\htdocs\
                ```

            - Abrir una terminal apuntando a esta dirección y ejecutar:
                ```bash
                git clone https://github.com/MARnVEL/saII_examenTFJS_24_05_velarde_presentado.git
                ```
        3. Abrir XAMPP
        4. Iniciar el servidor Apache
        5. En su navegador de preferencia escribir en la barra de búsqueda lo siguiente:
            ```bash
            http://localhost/saII_examenTFJS_24_05_velarde_presentado/t3_punto6.html
            ```

    - Opcion 2:
        - Una vez clonado el repositorio. Abrir el directorio con el IDE Visual Studio Code.
        - En Visual Studio Code instalar el plug-in ```Live Server``` que nos permite crear un servidor virtual.
        - Abrir el archivo ```t3_punto6.html```
        - Clic derecho sobre algún espacio en blanco de este fichero.
        - Clic en ```Open with Live Server (Abrir con Live Server)```
